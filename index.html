Intro

I am a theoretical physicist working in the field of Condensed matter physics. To describe my efforts as a researcher, I look for ways to stabilize a particular phase of spin systems known in the literature as 'spin liquids'. This includes the application of a combination of analytical and numerical techniques to predict certain desirable properties in spin systems. Analytical techniques include mean-field, linear and interacting spin-waves and quantum field theory in general. Numerical techniques entail Classical Monte Carlo, iterative minimization, variational Monte Carlo and exact diagonalization. We enjoy significant collaboration with the University of Illinois at Urbana Champaign for the numerical analysis. For a list of my publications, check out my google scholar link here.

Since the past 2 years, I have focused my attention on one of the most sought after fields since the last decade or so, Data Science. Leveraging my 8+ years of experience in quantitative sciences and through numerous online courses, I have been able to establish a somewhat strong foothold in the Data Science community. I have extensive hands-on experience with a multitude of data science projects. This includes both supervised and unsupervised tasks from a wide range of fields including but not limited to medical science, restaurant business, finance, computer vision, natural language processing,etc. On the deep learning side, I have also recently aced the tensorflow developer certification exam.

Projects

1. Segmentation of Los Angeles neighborhoods based on restaurant types/densities

Github link : IBM Capstone

As part of the capstone project for the IBM professsional certificate, I did an unsupervised learning probelm. The idea of the project is to explore restaurants in Los Angeles and predict optimal locations for new restaurant setups. For leveraging location data, I use Folium for producing maps indicating venue locations. For the learning part, I use a K-means clustering for segmenting the various neighborhoods according to restaurant densities, types or frequency of occurrence. Further details are in the github link.

2. Exploratory Data Analysis on a breast cancer dataset

Github link : Breast Cancer

In this project, I analyze a breast cancer dataset with seaborn plots. Based on such an analysis, a feature selection step is performed by filtering redundant correlated features along with eliminating some non-predictive features. The ML models then finally take advantage of this exploratory analysis.

3. Employee Churn Prediction

Github link : Churn prediction

This project deals with predicting employee churn using an interactive Decision tree and a Random Forest classifier. As part of a guided project in coursera, we used a Decision tree and a Random Forest model to predict employee churn. We also made the corresponding models interactive to visualize different Decsion tree/random forest models in one go. We also evaluated our model using the ROCAUC curves and compared it against a logistic regression model.

4. Tensorflow Projects

Github link : TensorFlow

This is a compilation of some of the deep learning projects that I worked on independently. This is by no means an exhaustive list. Its mostly for demonstration of the skills that I have acquired over the years as a learner and an enthusiast. The image classifiers below are also deployed to local browsers using Tensorflow.js

  • Beans Classifier

    • This model learns the Beans images dataset from tensorflow datasets. The images are first processed through an augmentation pipeline before being fed into training. This is all done using a tf.data pipeline for faster training. The trained model can take a picture of a bean leaf and predict whether it belongs to one of the two diseased categories - bean_rust or angular leaf spot or if its healthy.

  • Dog Breed Classifier

    • This model uses the stanford_dogs dataset available from tensorflow datasets to train a dog classifier. Like the Beans classifier, the images are again part of a tf.data training pipeline. Since this is a complicated dataset, we use transfer learning to train our model here. I use the InceptionV3 model with the default 'imagenet' weights. Inception is a highly optimized model for image classification benchmarked against the famous imagenet dataset. I set the last few layers of the model to be trainable for my purposes.

  • Sentiment Classifier

    • This model learns the yelp_polarity_reviews dataset from tensorflow datasets. This is a simple text classifier with 1D convolutional and global average pooling layers. It's a binary classifier with review polarities labelled as a negative '0' label and a positive '1' label. The common preprocessing steps for this text dataset was taken care of by the TextVectorization layer from tensorflow which is inherited from keras.layers class. Apart from vectorizing each text with intger tokens, this layer also limits the max_length of the reviews and also pads them whenever needed.
